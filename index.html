Un kit sencillo para crear un sitio tipo ChatGPT que genera apps sin código con OpenAI + Supabase + (plantillas iOS).

Inicio Rápido Cuentas Arquitectura Esquema Supabase Prompts API iOS / Xcode Deploy

Inicio Rápido

  1. Clona este proyecto o descomprime el ZIP.
  2. Crea un proyecto en Supabase y ejecuta el SQL en supabase/schema.sql y supabase/policies.sql.
  3. Configura variables de entorno (server): OPENAI_API_KEY, SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY.
  4. Levanta tu servidor (Node/Express o Next.js) y pega el ejemplo de code/api/openai.ts.
  5. Usa los prompts de sistema y el schema JSON para generar especificaciones de apps.
  6. Genera los archivos SwiftUI con node code/scripts/gen-ios.js (luego ábrelos en Xcode).

Cuentas y claves

  • OpenAI: API Key en OPENAI_API_KEY.
  • Supabase: SUPABASE_URL, SUPABASE_ANON_KEY (cliente) y SUPABASE_SERVICE_ROLE_KEY (server).
  • Apple Developer (opcional para TestFlight): Team ID, Issuer ID, Key ID, y Private Key.

Arquitectura

Web (Next.js/React) ─┬─> API /server
                      │      ├─ OpenAI (Diseño de AppSpec → UI → Código)
                      │      └─ Supabase (Auth, DB, Storage)
                      └─ Studio (builder visual + plantillas)
iOS (SwiftUI) ←─────── scripts/gen-ios.js (genera archivos base desde AppSpec JSON)

El flujo principal: Usuario describe → LLM genera AppSpec JSON → se guardan versiones en Supabase → (opcional) se generan stubs de UI iOS.

Esquema Supabase

Ejecuta los archivos supabase/schema.sql y supabase/policies.sql.

Prompts de Sistema & Esquema JSON

Usa los archivos en prompts/:

  • system-appspec.md: Diseñador de AppSpec (genera JSON.
  • system-uigen.md: Compositor de UI desde AppSpec.
  • system-guardrail.md: Validación/sanidad del JSON.
  • appspec.schema.json: Esquema de salida en JSON.

También puedes copiar el schema JSON directamente.

API (TypeScript/Node)

Ejemplo de endpoint que llama a OpenAI forzando JSON:

// code/api/openai.ts

Adapta esto a tu framework (Express/Next.js). Usa response_format: { type: "json_object" } y valida contra el schema.

Salida iOS (SwiftUI)

Genera archivos Swift desde AppSpec con:

node code/scripts/gen-ios.js ./appspec.json ./out-ios
  1. Abre Xcode → Crea un nuevo proyecto iOS con SwiftUI.
  2. Reemplaza ContentView.swift y agrega los archivos generados desde ./out-ios.

Deploy

  • Frontend: Vercel/Netlify.
  • Backend: Vercel Functions/Fly.io/Render.
  • DB/Auth/Storage: Supabase.
  • iOS: Xcode → TestFlight (Fastlane opcional).
